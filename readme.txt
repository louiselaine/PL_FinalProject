Installation Instructions:
--------------------------
1. Extract the folder and place it in your XAMPP 'htdocs' directory.
2. Start Apache and MySQL in XAMPP.
3. Open phpMyAdmin and create a new database (e.g., `teacher_db`).
4. Import the provided SQL file (e.g., `teacher_db.sql`) into the database.
5. Open the project in your browser:
   http://localhost/Teacher-Record-Management-System-php-project-main/
